#Weather Journal App

### Overview:

<hr>

Logs an entry with:

- Date
- Local current weather
- Current state of emotion

### Startup

<hr>

1. Clone my Repo
2. On your command-line run: `node server.js`
3. On your browser: visit (http://localhost:8000/)

### Technologies used

<hr>

- node.js
- Express.js
- Openweathermap API
